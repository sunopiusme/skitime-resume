## Почему self-host

- Build не зависит от сети к Google Fonts.
- Нет передачи IP-адресов посетителей третьим лицам при загрузке шрифтов.
- Контролируемое подмножество subsets и weights уменьшает payload.

При обновлении шрифтов сохраняйте файлы лицензий вместе с font-файлами и обновляйте этот README.
