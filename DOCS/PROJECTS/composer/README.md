# Composer

Первый portfolio-специальный проект `skitime-resume`: исследовательский кейс про ADE, Agentic Development Environment.

Скоуп v1 — editorial long-read `/projects/composer` с тремя живыми фрагментами (`ThinkingModel`, `ComposerInput`, `AgentControlPlane`), тремя research figures и парными code snippets.

Полноценное приложение, LLM backend, cloud sandbox, GitHub API и MCP-интеграции в скоуп не входят. Все agentic-поведения моделируются детерминированно.

## Карта чтения

1. `PRD.md` — продуктовый source of truth.
2. `DESIGN.md` — дизайн-контракт и handoff.
3. `ENGINEERING.md` — инженерная структура и quality gates.
