# UI Specification

## Visual Direction

- Tone: editorial, precise, product-grade.
- Avoid: template SaaS feel, utility-class visual sameness, purple-on-white defaults, decorative clutter.
- Palette:
  - `graphite` background
  - `warm paper` for primary type
  - `oxidized cyan` for signals
  - `muted copper` for secondary accent

## Typography

- Headlines MUST feel assertive and authored.
- Serif SHOULD be reserved for editorial emphasis and key narrative moments.
- Body copy MUST remain readable on mobile and print.

## Layout

- Landing SHOULD feel asymmetrical but stable.
- Cards MUST read as dossier panels rather than marketplace tiles.
- Resume page MUST support print without layout collapse.

## Motion

- Motion MAY exist only as subtle hover and load polish.
- Motion MUST NOT introduce layout instability or product ambiguity.
