# skitime-resume Documentation OS

Этот каталог является internal source-of-truth слоем проекта `skitime-resume`. Он нужен не как справка после факта, а как операционная рамка разработки.

## Нормативный язык

Ключевые слова `MUST`, `SHOULD`, `MAY` интерпретируются по BCP 14 / RFC 8174, когда записаны в верхнем регистре.

## Карта чтения

1. `DOCS/_SYSTEM/MANIFEST.md`
2. `DOCS/_SYSTEM/READING_ORDER.md`
3. `DOCS/PRODUCT/PRD.md`
4. `DOCS/ENGINEERING/TECH_SPEC.md`
5. `DOCS/DESIGN/UI_SPEC.md`
6. `DOCS/OPERATIONS/DELIVERY_CHECKLIST.md`
7. `DOCS/REFERENCES/SOURCE_MATRIX.md`

## Ownership

- Product truth живет в `PRODUCT/PRD.md`.
- Engineering truth живет в `ENGINEERING/TECH_SPEC.md`.
- UI truth живет в `DESIGN/UI_SPEC.md`.
- Delivery truth живет в `OPERATIONS/DELIVERY_CHECKLIST.md`.
- Source status живет в `REFERENCES/SOURCE_MATRIX.md`.
